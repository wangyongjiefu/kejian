01. Bootstrap4来了

Bootstrap4来了

概要

2018年1月28日，Bootstrap4终于来了，等得头发都白了不少。

CSS UI Framework

  • Bootstrap
  • jQuery UI
  • BootMetro(Windows8/10)
  • W3css
  • Flat UI
  • etc.

官方网站

http://getbootstrap.com/

使用版本

  • v4.0

基础知识

  • HTML5
  • CSS/CSS3
  • Javascript
  • jQuery

开发工具

  • 记事本等文本编辑器

    1. Visual Studio Code
    2. Brackets
    3. ATOM

  • 浏览器

    1. Chrome(推荐)
    2. FireFox
    3. IE/edge

视频计划

  • 每个视频只包括一个知识点，并控制在5分钟之内（甚至更短，但经常超时）
