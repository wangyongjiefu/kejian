# js运算符

> 算 字 赋 比 逻 位 它

### 算术运算符

```
 + - * / ++ --
```

### 字符串连接

```
 +
```

### 赋值运算

```
 =    +=   -=   %=
```

### 比较运算符

```
 <  >    >=    <=    ==    ===   !=   !==
```

### 逻辑运算符

```
 &&  ||  !
```

### 位运算

```
 ^    &    |   <<   >>
```

### 其它运算符

```
 ?   :    三元运算符
delete：用于删除对象中属性的   如：delete o.name; //删除o对象中的name属性
void：   void 运算符对任何值返回 undefined。没有返回值的函数真正返回的都是 undefined。
var iNum1=1, iNum2=2, iNum3=3;// 逗号运算符 用逗号运算符可以在一条语句中执行多个运算。
```
